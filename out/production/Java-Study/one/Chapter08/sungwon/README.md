        매개변수

6. 메소드 선언 시 아무 데이터도 리턴 타입으로 넘겨주지 않겠다는 것을 지정하는 키워드는 무엇인가요?

9. 기본 자료형을 매개 변수로 넘겨 줄 때 Pass by value인가요? 아니면 Pass by reference인가요?
